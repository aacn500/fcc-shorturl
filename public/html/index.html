Readme

Visit https://aacn500-reqheader.herokuapp.com/shorten with the url to be shortened as a query parameter (for example like this: https://aacn500-reqheader.herokuapp.com/shorten?url=www.google.com, and then get a JSON response containing the original url and your shortened url (something like https://aacn500-reqheader.herokuapp.com/aAbC). Then, navigate to the shortened url, and you will get redirected to your original url!


You could also shorten a url by pasting it below, and pressing 'Go!'!
